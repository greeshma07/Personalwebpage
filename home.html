Picture Montage

Project Description


This project was built using C++ and OpenCV. For this project, the input image was broken up into smaller blocks of arbitrary size and the average color in each of these blocks was calculated. A metric (based on the block and its average color) was used as a basis for replacing the block with a new image, from a database of possible images, which has approximately the same average color. You can find the source code in my Github page.
